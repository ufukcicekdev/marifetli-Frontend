# Bildirim ikonları (opsiyonel)

Push bildirimlerinde türe göre ikon göstermek için bu klasöre aşağıdaki PNG dosyalarını ekleyebilirsiniz (önerilen boyut: 96x96 veya 192x192):

- `notification-like.png` — beğeni
- `notification-comment.png` — yorum/cevap
- `notification-follow.png` — takip
- `notification-mention.png` — mention
- `notification-community.png` — topluluk daveti

Dosyalar yoksa varsayılan uygulama ikonu (`/android-chrome-192x192.png`) kullanılır.
