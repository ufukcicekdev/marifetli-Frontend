# Site yazı tipi (TT Octosquares)

Varsayılan site fontu **TT Octosquares** bu klasörden yüklenir.

## Dosya

- **TTOctosquares-Regular.woff2** — Bu dosyayı bu klasöre koyun.  
  (TypeType’tan lisanslı .woff2 dosyası; isim tam olarak `TTOctosquares-Regular.woff2` olmalı.)

Dosya yoksa tarayıcı sistem varsayılan fontuna düşer. Ek ağırlıklar (Medium, Bold vb.) için `globals.css` içindeki `@font-face` bloğuna ek `src` satırları ekleyebilirsiniz.
